Instruções para rodar o back-end

    --- PARA WINDOWS ---
    Baixa o node.js se na sua maquina não tiver
    acesse https://nodejs.org/ e instale a versão 'LTS' se for o caso.

    acesse a pasta script_ligarBackend

    Rode o Arquivo win_dependencias.bat para instalar as dependencias do projeto

    Para iniciar o servidor back-end:
    Rode o Arquivo win_modo_admin.bat para abilitar publicação e edição de projetos
    ou
    Rode o Arquivo win_modo_normal.bat para desabilitar publicação e edição de projetos

    e depois entre na url:http://localhost:3000 

    pra para de rodar é so fechar a janela


    --- PARA LINUX ---
    Baixa o node.js se na sua maquina não tiver
    Abra o terminal de comando e copie os seguintes comandos para baixar o node js:

    wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
    export NVM_DIR="$HOME/.nvm"
    [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

    e confirar se funcionou com esse comando: node -v

    caso apareça algo como: v22.19.0 
    deu tudo certo

    Para iniciar o servidor back-end:
    Rode o Arquivo Lx_modo_admin.bat para abilitar publicação e edição de projetos
    ou
    Rode o Arquivo Lx_modo_normal.bat para desabilitar publicação e edição de projetos

    e depois entre na url:http://localhost:3000 

    pra para de rodar é so fechar a janela


    --- para rodar no Vscode ---

    npm install no terminal cmd

    depois 

    node src/server.js

    Ctrl + C no terminal para para de Rodar
